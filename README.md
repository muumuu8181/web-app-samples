## 🚀 使用方法

### オンラインで試す
各アプリケーションのデモリンクからすぐに体験できます。

### ローカルで実行
```bash
# リポジトリをクローン
git clone https://github.com/muumuu8181/web-app-samples.git
cd web-app-samples

# 特定のアプリを実行（例：色彩検定クイズ）
cd color-quiz-app
python -m http.server 8000

# ブラウザで開く
open http://localhost:8000
```
